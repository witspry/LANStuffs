LanStuffs

Introduction :

The application LanStuffs is all about connecting two or more individual LanStuffs applications located in different computers connected via LAN (Local Area Network). This application uses WinSock (Windows Socket) to connect and communicate between these two different applications and provide an illusion or virtualization of pure peer-to-peer networking communication which is generally not possible using WinSocks.


To connect and communicate between these two applications, please follow the below steps:

1) Run LanStuffs application on different computers (connected through LAN) on which you want to communicate.

2) Click on the 'Connect' button. This will create a application server on default port number '1234' which actually receive the incoming request and will provide to the application for processing.

3) After doing the second step with all the applications, goto :- Tools -> Options -> Manage Connections. In this window you can add connections to other applications running on different computers. You just have to provide the IP Addresses ( Internet Protocol Adress e.g. "192.167.23.87") of other computer connected through LAN.

4) You have to do the above steps with all the applications that you want to communicate.

5) Now, the connection setup is finished.

I the similar way, I can send a file from my computer to all other connected computer without sharing the drives and folders.

*One of the most vital featue of this product is its Security. All the text messages and files that are sent through this application are completely secure as it uses a 256 bit key strong encryption supported by one of the most secure algorithm that we know - "Triple Data Encryption Standard (3 DES) ". Remember - the whole data is encrypted before sending.
